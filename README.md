# Music-Bot

Once authorized on a discord channel. The Bot listens for users requests and plays songs accordingly.
It has the capability to Play, Pause, and to create a queue of songs

<h2> Usage </h2>
<ol>
  <li> !play SongName  </li> 
  <li> !stop </li>
  <li> !skip </li>
  <li> !pause </li>
  <li> !skip </li>
  <li> random </li>
 </ol>
